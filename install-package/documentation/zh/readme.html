有关常见问答、已知问题和修订的列表以及其他支持信息，请访问 IBM Installation Manager 支持门户网站。

有关产品新闻、事件和其他信息，请访问 IBM Rational® 软件 Web 站点。

在您联系 IBM Rational 软件支持之前，请收集描述问题所需要的背景信息。向 IBM 软件支持专家描述问题时，请尽量具体，要包括所有相关背景信息，以便专家可以帮助您有效地解决问题。要节省时间，请了解这些问题的答案：

  • 发生问题时您运行的软件版本是什么？
  • 您具有与问题相关的日志、跟踪或消息吗？
  • 您可以重现该问题吗？如果可以，那么重现该问题需要哪些步骤？
  • 该问题存在变通方法吗？
  • 如果存在，请准备描述变通方法。
