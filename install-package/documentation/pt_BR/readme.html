Problemas Conhecidos

Os problemas conhecidos são registrados em documentos individuais na Base de Conhecimento de Suporte. À medida que os problemas são descobertos e resolvidos, a base de conhecimento é atualizada e mantida com novas informações. Procurando na base de conhecimento, é possível localizar rapidamente soluções alternativas ou soluções para problemas.

IBM Rational Software Support

Para obter informações de contato e diretrizes ou materiais de referência que você precisará ao requerer suporte, leia o IBM Software Support Handbook.

Para consultar as perguntas mais frequentes, listas de problemas e correções conhecidos e outras informações de suporte, visite o IBM Installation Manager Portal de suporte.

Para notícias de produto, eventos e outras informações, visite o website do IBM Rational® Software.

Antes de entrar em contato com o Suporte IBM Rational Software, reúna as informações que serão necessárias para descrever seu problema. Ao descrever um problema para um especialista de suporte de software IBM, seja o mais específico possível e inclua todas as informações relevantes para que o especialista possa ajudar a resolver o problema de maneira eficiente. Para economizar tempo, saiba as respostas a estas perguntas:

  • Quais versões de software você estava executando quando o problema ocorreu?
  • Você tem logs, rastreios ou mensagens que estão relacionados ao problema?
  • Você consegue reproduzir o problema? Se sim, quais etapas você executa para reproduzi-lo?
  • Há uma solução alternativa para o problema?
  • Se sim, esteja preparado para descrever a solução alternativa.
