คำอธิบาย

สำหรับรายละเอียด เกี่ยวกับ IBM Installation Manager, โปรดดูที่ ศูนย์ข้อมูล IBM Installation Manager

ข้อกำหนดของระบบ

สำหรับ ข้อมูลเกี่ยวกับข้อกำหนดด้านฮาร์ดแวร์และซอฟต์แวร์สำหรับการติดตั้ง ผลิตภัณฑ์ โปรดดูที่ ข้อกำหนดระบบสำหรับ IBM Installation Manager

คำแนะนำในการติดตั้ง

สำหรับ ขั้นตอนการติดตั้งแบบทีละขั้น โปรดดูที่ คำแนะนำการติดตั้ง สำหรับ Installation Manager หรือ อ้างอิง IBM Knowledge Center สำหรับผลิตภัณฑ์ของ IBM ที่คุณกำลังติดตั้งอยู่

สำหรับคำแนะนำเกี่ยวกับการติดตั้งฟิกซ์แพ็กนี้ไม่ว่าจะเป็นการอัพเดต หรือในเวลาเดียวกับที่คุณติดตั้งผลิตภัณฑ์พื้นฐาน โปรดดูที่ ดาวน์โหลด IBM Installation Manager

สำหรับคำแนะนำเกี่ยวกับการติดตั้ง IBM Software Update Notifier โปรดดูที่ เอกสารคู่มือการดาวน์โหลด IBM Software Update Notifier

ปัญหาที่พบ

ปัญหาที่พบ จะถูกบันทึกไว้ในเอกสารต่างหากใน ฐานความรู้การสนับสนุน ขณะที่ปัญหาถูกค้นพบ และแก้ไข ฐานความรู้จะถูกอัพเดต และได้รับการดูแลรักษาด้วยข้อมูลใหม่ โดย การค้นหาฐานความรู้ คุณสามารถค้นหาวิธีแก้ปัญหา หรือ โซลูชันสำหรับปัญหาได้อย่างรวดเร็ว

การสนับสนุนซอฟต์แวร์ IBM Rational

สำหรับข้อมูลการติดต่อและแนวปฏิบัติ หรือเอกสารอ้างอิง ที่คุณจำเป็นต้องใช้เมื่อต้องการการสนับสนุน โปรดอ่านที่ การสนับสนุนซอฟต์แวร์ของ IBM

สำหรับ FAQ, รายการปัญหาที่พบและ โปรแกรมฟิกซ์ และข้อมูลการสนับสนุนอื่นๆ โปรดเยี่ยมชมที่ พอร์ทัลการสนับสนุนของ IBM Installation Manager

สำหรับข่าวสารเกี่ยวกับผลิตภัณฑ์ กิจกรรม และข้อมูลอื่นๆ โปรดเยี่ยมชมที่ เว็บไซต์ซอฟต์แวร์ IBM Rational®

ก่อนที่คุณจะติดต่อฝ่ายสนับสนุนซอฟต์แวร์ IBM Rational โปรดรวบรวมข้อมูลเบื้องต้นที่คุณจำเป็นต้องอธิบายถึง ปัญหาของคุณ เมื่ออธิบายถึงปัญหาให้กับผู้เชี่ยวชาญด้านการสนับสนุน ซอฟต์แวร์ IBM โปรดระบุถึงปัญหาให้ชัดเจน และบรรยายข้อมูลเบื้องต้น ที่เกี่ยวข้องทั้งหมด เพื่อให้ผู้เชี่ยวชาญสามารถช่วยคุณแก้ไขปัญหา ได้อย่างมีประสิทธิภาพ เพื่อเป็นการประหยัดเวลา ควรหาคำตอบเกี่ยวกับคำถามเหล่านี้ไว้ล่วงหน้า:

  • เวอร์ชันของซอฟต์แวร์ที่คุณใช้รันเมื่อเกิดปัญหาคือเวอร์ชันอะไร
  • คุณมีไฟล์บันทึก, เธรด หรือข้อความที่เกี่ยวข้องกับ ปัญหาหรือไม่
  • คุณสามารถทำให้เกิดปัญหาซ้ำอีกครั้งได้หรือไม่ หากทำได้ ขั้นตอนใดที่คุณใช้เพื่อ ทำให้เกิดปัญหาอีกครั้ง
  • มีวิธีแก้ไขปัญหาหรือไม่
  • หากมี ให้เตรียมเพื่ออธิบายวิธีแก้ปัญหา
