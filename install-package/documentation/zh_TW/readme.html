說明

系統需求

如需安裝本產品的軟硬體需求相關資訊，請參閱 IBM Installation Manager 的系統需求。

安裝指示

如需逐步安裝指示，請參閱 Installation Manager 的安裝手冊，或參閱您正在安裝之 IBM 產品的 IBM Knowledge Center。

如需作為更新項目或在安裝基本產品的同時，安裝此修正套件的相關指示，請參閱 IBM Installation Manager 下載。

已知問題

已知問題會以個別的文件記載於支援中心知識庫中。隨著我們不斷地發現與解決問題，將會以新資訊來更新及維護知識庫。透過搜尋知識庫，即可快速找到問題的暫行解決方法或解決方案。

IBM Rational 軟體支援中心

關於在需要支援時所需的聯絡資訊和準則或參考資料，請閱讀《IBM 軟體支援手冊》。

如需常見問題、已知問題與修正程式清單及其他支援資訊，請造訪 IBM Installation Manager 支援中心入口網站。

如需產品新訊、事件及其他資訊，請造訪 IBM Rational® 軟體網站。

在聯絡「IBM Rational 軟體支援中心」之前，請收集說明問題所需的背景資訊。在向 IBM 軟體支援專家說明問題時，請儘可能詳細地加以說明，並提供所有相關背景資訊，以便專家能協助您有效地解決問題。為了節省時間，請找出下列問題的答案：

  • 發生問題時，您正在執行哪個軟體版本？
  • 是否擁有該問題相關的日誌、追蹤資料或訊息？
  • 能否重現該問題？若可以重現，需要執行哪些步驟？
  • 是否有該問題的暫行解決方法？
  • 若有，請準備說明暫行解決方法。
