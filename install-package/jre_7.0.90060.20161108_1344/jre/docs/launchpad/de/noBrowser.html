Beim Starten des Launchpads ist ein Fehler aufgetreten. Dieser Fehler tritt in der Regel auf, wenn das Launchpad keinen unterstützten Browser finden kann.

Zugriff auf die Dokumentation ohne das Launchpad
=================================================

Sie können ohne Browser auf die Dokumentation zugreifen, indem Sie das Verzeichnis 'docs' von SDK verwenden. Die Dokumentationsdateien sind im Verzeichnis 'content' enthalten.

Die API-Dokumentation ist im Verzeichnis 'apidocs' enthalten. Weitere Dokumentation ist in sprachspezifischen Verzeichnissen enthalten. Falls ein Dokument nicht in der gewünschten Sprache vorhanden ist, verwenden Sie die englische Version des Dokuments im Verzeichnis 'en'.

Beheben des Fehlers, auf Grund dessen die Anzeige des Launchpads nicht funktioniert
=====================================================================

Das Launchpad unterstützt die folgenden Browser:
o Mozilla
o Firefox
o Internet Explorer (nur Microsoft Windows-Plattformen)

Versuchen Sie mit der folgenden Prozedur, den Fehler zu beheben, auf Grund dessen die Anzeige des Launchpads nicht funktioniert, und versuchen Sie, das Launchpad erneut zu starten:

1. Falls Sie nicht über den Web-Browser Mozilla verfügen, laden Sie ihn von der Website 'http://www.mozilla.org' herunter, und installieren Sie ihn.

Exportieren Sie auf Linux- oder UNIX-Plattformen die Position des unterstützten Browsers. Beispiel:

export BROWSER=/usr/bin/mozilla

2. Starten Sie das Launchpad mit dem folgenden Befehl erneut:

Linux- und UNIX-Plattformen: ./launchpad.sh
Windows-Plattformen: launchpad.exe
