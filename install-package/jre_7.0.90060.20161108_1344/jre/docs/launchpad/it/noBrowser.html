Si è verificato un errore durante l'avvio del launchpad. Tale errore si verifica generalmente quando il launchpad non è in grado di trovare un browser supportato.

Accesso alla documentazione senza il launchpad
=================================================

È possibile accedere alla documentazione senza un browser visualizzando la directory docs di sdk. I file della documentazione si trovano nella directory content.

La documentazione API si trova nella directory apidocs. La restante documentazione si trova nelle directory specifiche per le lingue. Se un documento non esiste per la lingua desiderata, utilizzare la versione inglese nella directory en.

Correzione dell'errore che impedisce la visualizzazione del launchpad
=====================================================================

Il launchpad supporta i seguenti browser:
o Mozilla
o Firefox
o Internet Explorer (solo piattaforme Microsoft Windows)

Utilizzare la seguente procedura per tentare di correggere l'errore che impedisce la visualizzazione del launchpad e cercare di riavviare il launchpad:

1. se non si dispone del browser Web Mozilla, scaricare ed installare il browser Web Mozilla da http://www.mozilla.org.

Su Linux e piattaforme UNIX, esportare la posizione del browser supportato. Ad esempio:

esportare BROWSER=/usr/bin/mozilla

2. Riavviare il launchpad immettendo il seguente comando:

Linux e piattaforme UNIX: ./launchpad.sh
Piattaforme Windows: launchpad.exe
