런치패드를 시작하는 중에 오류가 발생했습니다. 이 오류는 일반적으로 런치패드가 지원되는 브라우저를 찾을 수 없는 경우에 발생합니다.

런치패드 없이 문서에 액세스
=================================================

브라우저가 없는 경우 SDK의 docs 디렉토리의 문서에 액세스할 수 있습니다. 문서 파일은 content 디렉토리에 있습니다.

API 문서는 apidocs 디렉토리에 있으며 기타 문서는 language-specific 디렉토리에 있습니다. 해당 언어의 문서가 없는 경우, en 디렉토리에 있는 영어 버전을 사용하십시오.

런치패드 표시를 방해하는 오류 정정
=====================================================================

런치패드는 다음 브라우저를 지원합니다.
o Mozilla
o Firefox
o Internet Explorer(Microsoft Windows 플랫폼 전용)

다음 프로시저를 사용하여 런치패드 표시를 방해하는 오류를 정정하고 런치패드를 다시 시작하십시오.

1. Mozilla 웹 브라우저가 없는 경우, http://www.mozilla.org에서 Mozilla 웹 브라우저를 다운로드하여 설치하십시오.

Linux 및 UNIX 플랫폼에서, 지원되는 브라우저의 위치를 내보내십시오. 예를 들면 다음과 같습니다.

export BROWSER=/usr/bin/mozilla

2. 다음 명령을 실행하여 런치패드를 다시 시작하십시오.

Linux 및 UNIX 플랫폼: ./launchpad.sh
Windows 플랫폼: launchpad.exe
