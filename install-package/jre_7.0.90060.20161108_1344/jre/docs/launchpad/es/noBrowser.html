Se ha producido un error al iniciar el Launchpad. Este error suele ocurrir cuando el Launchpad no puede encontrar un navegador soportado.

Acceso a la documentación sin el Launchpad
=================================================

Puede acceder a la documentación sin un navegador examinando el directorio docs del SDK. Los archivos de documentación están en el directorio content.

La documentación de API está en el directorio apidocs. El resto de documentación está en los directorios específicos de cada idioma. Si un documento no existe para su idioma, utilice la versión en inglés del directorio en.

Corrección del error que está impidiendo que se visualice el Launchpad
=====================================================================

El Launchpad da soporte a los siguientes navegadores:
o Mozilla
o Firefox
o Internet Explorer (sólo plataformas Microsoft Windows)

Utilice el siguiente procedimiento para intentar corregir el error que está impidiendo que se visualice el Launchpad y para intentar restaurar el Launchpad:

1. Si no tiene el navegador web Mozilla, descargue e instale el navegador web Mozilla desde http://www.mozilla.org.

En las plataformas Linux y UNIX, exporte la ubicación del navegador soportado. Por ejemplo:

export BROWSER=/usr/bin/mozilla

2. Reinicie el Launchpad emitiendo el mandato siguiente:

Plataformas Linux y UNIX: ./launchpad.sh
Plataformas Windows: launchpad.exe
