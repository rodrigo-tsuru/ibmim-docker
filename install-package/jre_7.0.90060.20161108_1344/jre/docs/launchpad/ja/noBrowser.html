ランチパッドを開始中にエラーが発生しました。このエラーは通常、サポートされるブラウザーをランチパッドが検出できない場合に発生します。

ランチパッドを使用しない資料の利用
=================================================

SDK の docs ディレクトリーを調べることによって、ブラウザーを使用せずに資料にアクセスできます。資料ファイルは、content ディレクトリーにあります。

API 資料は、apidocs ディレクトリーにあります。その他の資料は、言語固有のディレクトリーにあります。ご使用の言語の文書が存在しない場合は、en ディレクトリー内の英語バージョンを使用してください。

ランチパッドが表示されない原因となっているエラーの修正
=====================================================================

ランチパッドは以下のブラウザーをサポートします。
o Mozilla
o Firefox
o Internet Explorer (Microsoft Windows プラットフォームのみ)

以下の手順に従って、ランチパッドが表示されない原因となっているエラーの訂正を試み、ランチパッドを再始動してください。

1. Mozilla Web ブラウザーを所有していない場合は、http://www.mozilla.org から Mozilla Web ブラウザーをダウンロードし、インストールします。

Linux および UNIX プラットフォームでは、サポートされるブラウザーのロケーションを export します。例えば、次のようにします。

export BROWSER=/usr/bin/mozilla

2. 次のコマンドを実行して、ランチパッドを再始動します。

Linux および UNIX プラットフォーム: ./launchpad.sh
Windows プラットフォーム: launchpad.exe
