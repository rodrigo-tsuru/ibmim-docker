不使用启动板来访问文档
=================================================

您可以不通过浏览器，以直接查看 SDK 文档目录的方式来访问文档。文档文件存放在 content 目录下。

API 文档存放在 apidocs 目录下。其他文档各自存放在特定于语言的目录下。如果不存在使用您的语言的文档，请使用 en 目录下的英语版本。

纠正使启动板无法显示的错误
=====================================================================

启动板支持下列浏览器：
o Mozilla
o Firefox
o Internet Explorer（仅 Microsoft Windows 平台）

使用下列过程，尝试纠正使启动板无法显示的错误，然后重新启动启动板：

1. 如果没有 Mozilla Web 浏览器，请从 http://www.mozilla.org 下载并安装 Mozilla Web 浏览器。

在 Linux 和 UNIX 平台，导出受支持浏览器的位置。例如：

export BROWSER=/usr/bin/mozilla

2. 发出下列命令重新启动启动板：

Linux 和 UNIX 平台：./launchpad.sh
Windows 平台：launchpad.exe
