Une erreur s'est produite lors du démarrage du tableau de bord. Cette erreur se produit généralement lorsque le tableau de bord ne parvient pas à trouver de navigateur pris en charge.

Accès à la documentation sans le tableau de bord
=================================================

Vous pouvez accéder à la documentation sans navigateur en accédant au répertoire des documents du sdk. Les fichiers de documentation sont conservés dans le répertoire content.

La documentation de l'API est conservée dans le répertoire apidocs. Les autres documents sont conservés dans les répertoires spécifiques à chaque langue. Si un document n'existe pas pour votre langue, utilisez la version anglaise, dans le répertoire en.

Correction de l'erreur empêchant l'affichage du tableau de bord
=====================================================================

Le tableau de bord prend en charge les navigateurs suivants :
o Mozilla
o Firefox
o Internet Explorer (plateformes Microsoft Windows uniquement)

Utilisez la procédure suivante pour tenter de corriger l'erreur empêchant l'affichage du tableau de bord et essayer de redémarrer ce dernier :

1. Si vous ne disposez pas du navigateur Web Mozilla, téléchargez-le et installez-le à partir du site suivant : http://www.mozilla.org.

Sur les plateformes Linux et UNIX, exportez l'emplacement du navigateur pris en charge. Par exemple :

export BROWSER=/usr/bin/mozilla

2. Redémarrez le tableau de bord à l'aide de la commande suivante :

Plateformes Linux et UNIX : ./launchpad.sh
Plateformes Windows : launchpad.exe
