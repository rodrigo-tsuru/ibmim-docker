Ocorreu um erro ao iniciar a barra de ativação. Normalmente esse erro ocorre quando a barra de ativação não consegue localizar um navegador suportado.

Acessando a documentação sem a barra de ativação
=================================================

Você pode acessar a documentação sem um navegador consultando o diretório docs do sdk. Os arquivos de documentação ficam no diretório de conteúdo.

A documentação da API fica no diretório apidocs. Outras documentações ficam nos diretórios específicos dos idiomas. Se um documento não existir no seu idioma, utilize a versão em inglês no diretório en.

Corrigindo o erro que está impedindo a exibição da barra de ativação
=====================================================================

A barra de ativação suporta os seguintes navegadores:
o Mozilla
o Firefox
o Internet Explorer (apenas plataformas Microsoft Windows)

Utilize o seguinte procedimento para tentar corrigir o erro que está impedindo a exibição da barra de ativação e tente reiniciar a barra de ativação:

1. Se você não tiver um navegador da Web Mozilla, faça download e instale esse navegador a partir do site http://www.mozilla.org.

Em plataformas Linux e UNIX, exporte o local do navegador suportado. Por exemplo:

export BROWSER=/usr/bin/mozilla

2. Reinicie a barra de ativação emitindo o seguinte comando:

Plataformas Linux e UNIX: ./launchpad.sh
Plataformas Windows: launchpad.exe
