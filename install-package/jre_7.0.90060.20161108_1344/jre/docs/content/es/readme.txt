IBM SDK para Linux, Java Technology Edition, versión 7
=====================================================

Este archivo README se aplica a la versión 7, y a todos los releases, modificaciones y renovaciones de servicio siguientes hasta que se indique lo contrario
en un nuevo archivo README.

IBM proporciona contenido adicional con el SDK.

Documentación del producto:
----------------------

Las guías de usuario para dar soporte a IBM SDK para Java 7 están disponibles para la visualización en línea
en un Information Center de IBM: http://publib.boulder.ibm.com/infocenter/java7sdk/v7r0/index.jsp.

Estas guías también están disponibles para su descarga en formato de plug-in de PDF y Eclipse
desde IBM developerWorks: https://www.ibm.com/developerworks/java/jdk/docs.html.

También se puede encontrar documentación del producto adicional, incluida documentación de la API y de seguridad,
en developerWorks en el URL que se muestra.

Se puede encontrar cualquier noticia de última hora en la siguiente nota técnica de soporte de IBM:
http://www.ibm.com/support/docview.wss?uid=swg21499721
