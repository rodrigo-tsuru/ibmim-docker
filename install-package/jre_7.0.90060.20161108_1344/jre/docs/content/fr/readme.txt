IBM SDK for Linux, Java Technology Edition, Version 7
=====================================================

Ce fichier README s'applique à la version 7 et à toutes les versions, modifications et actualisations de service
suivantes, sauf indication contraire dans une nouvelle édition du fichier.

IBM fournit du contenu supplémentaire avec le kit de développement de logiciels.

Documentation du produit :
----------------------

Les guides d'utilisation pour la prise en charge d'IBM SDK for Java 7 sont disponibles en ligne dans le centre de documentation IBM http://publib.boulder.ibm.com/infocenter/java7sdk/v7r0/index.jsp.

Ces guides peuvent être également téléchargés dans les formats PDF et plug-in Eclipse depuis IBM developerWorks sur le site https://www.ibm.com/developerworks/java/jdk/docs.html.

Il existe également une documentation supplémentaire sur le produit, y compris sur la sécurité et les API, sur developerWorks accessible à l'URL indiquée.

Les dernières informations sont disponibles dans la note technique IBM
http://www.ibm.com/support/docview.wss?uid=swg21499721
