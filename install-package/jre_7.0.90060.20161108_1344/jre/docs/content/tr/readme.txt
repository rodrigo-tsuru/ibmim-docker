IBM SDK for Linux, Java Technology Edition, Sürüm 7
===================================================

Bu BENİ OKU dosyası, Sürüm 7 için ve yeni bir BENİ OKU dosyasında tersi
belirtilmediği sürece, sonraki tüm yayınlar, değişiklikler ve hizmet
