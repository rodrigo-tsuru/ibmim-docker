IBM SDK per a Linux, Java Technology Edition, Versió 7
======================================================

Aquest fitxer README s'aplica a la versió 7 i a totes les versions, modificacions
i actualitzacions de servei subsegüents, fins que s'indiqui el contrari en un nou
fitxer README.

IBM proporciona continguts addicionals amb el SDK.

Documentació del producte:
--------------------------

Podeu trobar guies d'usuari per donar suport a l'IBM SDK for Java 7 per visualitzar-les en línia
en un Information Center d'IBM: http://publib.boulder.ibm.com/infocenter/java7sdk/v7r0/index.jsp.

Aquestes guies també estan disponibles per a baixar-les en format PDF i de connector d'Eclipse
a IBM developerWorks: https://www.ibm.com/developerworks/java/jdk/docs.html.

A developerWorks, a l'URL indicat, també es pot trobar documentació addicional del producte,
inclosa documentació de seguretat i de l'API. 

Les notícies més recents es poden trobar a la següent nota tècnica de suport d'IBM:
http://www.ibm.com/support/docview.wss?uid=swg21499721
