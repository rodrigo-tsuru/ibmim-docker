IBM SDK for Linux, Java Technology Edition V7
=============================================

Ez a README a 7-es változatra és minden azt követő kiadásra, módosításra és
szervizfrissítésre vonatkozik, amíg azt egy új README fájlban
másként nem jelzik.

Az IBM további tartalmat biztosít az SDK-val.

Termékdokumentáció:
-------------------

Az IBM SDK for Java 7 felhasználói kézikönyvei online megtekinthetők az
IBM információs központban: http://publib.boulder.ibm.com/infocenter/java7sdk/v7r0/index.jsp.

A kézikönyvek PDF formátumban és Eclipse bedolgozóként is letölthetők az
IBM developerWorks webhelyről: https://www.ibm.com/developerworks/java/jdk/docs.html.

A developerWorks fenti URL címén további termékdokumentáció is elérhető,
beleértve a biztonsági és API dokumentációkat is.

A legfrissebb híreket a következő IBM támogatási technikai feljegyzés tartalmazza:
http://www.ibm.com/support/docview.wss?uid=swg21499721
