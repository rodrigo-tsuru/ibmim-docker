IBM SDK para Linux, Java Technology Edition, Versão 7
=====================================================

Este arquivo LEIA-ME se aplica à Versão 7 e a todas as liberações
subsequentes, modificações e atualizações de serviços, até que seja indicado de outra maneira
em um novo arquivo LEIA-ME.

A IBM fornece conteúdo adicional com o SDK.

Documentação do produto:
------------------------

Os guias do usuário para suporte ao IBM SDK para Java 7 estão disponíveis para visualização on-line
em um Centro de Informações IBM: http://publib.boulder.ibm.com/infocenter/java7sdk/v7r0/index.jsp.

Esses guias também estão disponíveis para download em formato PDF e de plug-in do Eclipse
no IBM developerWorks: https://www.ibm.com/developerworks/java/jdk/docs.html.

A documentação adicional do produto, incluindo a documentação de segurança e da API também
pode ser localizada no developerWorks na URL exibida.

Quaisquer novidades posteriores podem ser localizadas na seguinte nota técnica da IBM:
http://www.ibm.com/support/docview.wss?uid=swg21499721
