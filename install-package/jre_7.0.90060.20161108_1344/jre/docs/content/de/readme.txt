IBM SDK for Linux, Java Technology Edition, Version 7
=====================================================

Diese Readme-Datei bezieht sich auf Version 7 und alle nachfolgenden Releases
und Modifikationen, bis dieser Hinweise in einer neuen Readme-Datei geändert
wird.

IBM stellt mit dem SDK Dokumentation bereit.

Produktdokumentation:
---------------------

Unterstützende Benutzerhandbücher zu IBM SDK for Java 7 können in einem
IBM Information Center unter der folgenden Adresse online angezeigt werden:
http://publib.boulder.ibm.com/infocenter/java7sdk/v7r0/index.jsp.

Diese Handbücher können auch im PDF-Format oder als Eclipse-Plug-ins von
IBM developerWorks heruntergeladen werden:
https://www.ibm.com/developerworks/java/jdk/docs.html.

Über die oben genannte URL von developerWorks kann auch weitere
Produktdokumentation, einschließlich der Dokumentation zur Sicherheit
und zur API heruntergeladen werden.

Die aktuellen News finden Sie im folgenden technischen Hinweis des IBM Support:
http://www.ibm.com/support/docview.wss?uid=swg21499721
