IBM SDK for Linux, Java Technology Edition, wersja 7
====================================================

Ten plik README dotyczy wersji 7 i wszystkich kolejnych wydań, modyfikacji
i poprawek serwisowych aż do odwołania w nowym pliku README.

Firma IBM udostępnia dodatkową zawartość w pakiecie SDK.

Dokumentacja produktu:
----------------------

Podręczniki użytkownika produktu IBM SDK for Java 7 są dostępne w Centrum
informacyjnym IBM pod adresem:
http://publib.boulder.ibm.com/infocenter/java7sdk/v7r0/index.jsp.

Podręczniki te dostępne są również w postaci dostępnych do pobrania plików PDF
oraz jako wtyczki Eclipse w serwisie IBM developerWorks pod adresem:
https://www.ibm.com/developerworks/java/jdk/docs.html.

Dodatkowe informacje na temat produktu, w tym dokumentacja zabezpieczeń
i funkcji API, dostępne są w serwisie developerWorks pod podanym wyżej adresem.

Najnowsze informacje można znaleźć w nocie technicznej IBM:
http://www.ibm.com/support/docview.wss?uid=swg21499721
