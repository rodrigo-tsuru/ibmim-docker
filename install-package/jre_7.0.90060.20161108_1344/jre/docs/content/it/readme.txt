IBM SDK per Linux, Java Technology Edition, Versione 7
=====================================================

Questo file README si applica alla Versione 7 e a tutte le release, modifiche e
aggiornamenti di servizio successivi, fino a quando non diversamente indicato
in un nuovo file README.

L'IBM fornisce ulteriore contenuto con SDK.

Documentazione del prodotto:
----------------------

Le guide dell'utente che supportano IBM SDK per Java 7 sono disponibili per la visualizzazione in linea
in un Centro informazioni IBM all'indirizzo: http://publib.boulder.ibm.com/infocenter/java7sdk/v7r0/index.jsp.

Queste guide sono disponibili anche per il download in formato plug-in Eclipse e PDF
su IBM developerWorks: https://www.ibm.com/developerworks/java/jdk/docs.html.

Inoltre, è possibile trovare ulteriore documentazione del prodotto, inclusa la documentazione dell'API e di sicurezza, su
developerWorks all'URL menzionato.

È possibile trovare eventuali ultime notizie nella seguente nota tecnica dell'assistenza IBM:
http://www.ibm.com/support/docview.wss?uid=swg21499721
