IBM SDK for Linux, Java Technology Edition, Version 7
=====================================================

Этот файл README относится к версии 7 и всем последующим выпускам,
модификациям и пакетам обновлений, если иное не оговорено в
новом файле README.

Вместе с SDK компания IBM предоставляет дополнительные материалы.

Документация по продукту:
-------------------------

Руководства пользователей, дополняющие IBM SDK for Java 7, доступны в электронном виде
на веб-сайте IBM Information Center: http://publib.boulder.ibm.com/infocenter/java7sdk/v7r0/index.jsp.

Эти руководства также можно загрузить в формате PDF или формате модуля Eclipse
с IBM developerWorks: https://www.ibm.com/developerworks/java/jdk/docs.html.

Дополнительная документация по продукту, включая документацию по безопасности и API,
также доступна на developerWorks по указанному адресу.

Последние важные новости публикуются в технических заметках службы поддержки IBM:
http://www.ibm.com/support/docview.wss?uid=swg21499721
