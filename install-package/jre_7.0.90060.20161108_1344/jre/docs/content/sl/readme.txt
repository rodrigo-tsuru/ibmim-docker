IBM SDK for Linux, Java Technology Edition, različica 7
=====================================================

Ta datoteka PREBERI velja za različico 7 in za vse nadaljnje izdaje, spremembe in osvežitve storitev, dokler to ni naznačeno v novi datoteki PREBERI.

IBM s SDK-jem nudi dodatno vsebino.
Dokumentacija izdelka:
----------------------

Vodiči za uporabnike za podporo izdelka IBM SDK for Java 7 so za spletno ogledovanje na voljo v IBM-ovem informacijskem centru: http://publib.boulder.ibm.com/infocenter/java7sdk/v7r0/index.jsp.
Te vodiče lahko prek strani IBM developerWorks prenesete tudi v formatu PDF ali v formatu vtičnika Eclipse: https://www.ibm.com/developerworks/java/jdk/docs.html. Dodatno dokumentacijo izdelka, vključno z dokumentacijo varnosti in dokumentacijo API-jev, lahko prav tako najdete na prikazanem URL-ju strani IBM developerWorks. Vse kasnejše pomembne novice lahko najdete v naslednjem tehničnem obvestilu IBM-ove službe za podporo:
http://www.ibm.com/support/docview.wss?uid=swg21499721
